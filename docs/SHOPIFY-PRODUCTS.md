## PASO 4: Anadir Mas Productos (Opcional)

Para anadir mas productos que se traduzcan automaticamente, necesitas:

1. **Crear el producto en Shopify** con el handle correcto
2. **Anadir las traducciones** en `lib/seo-data.ts` bajo `PRODUCT_NAMES`

### Ejemplo para anadir "Silla de Ducha Giratoria":

\`\`\`typescript
// En lib/seo-data.ts, dentro de PRODUCT_NAMES:
export const PRODUCT_NAMES: Record<Locale, Record<string, string>> = {
  es: {
    // ... productos existentes
    'silla-ducha-giratoria': 'Silla de Ducha Giratoria',
  },
  en: {
    // ... productos existentes
    'silla-ducha-giratoria': 'Swivel Shower Chair',
  },
  de: {
    // ... productos existentes
    'silla-ducha-giratoria': 'Drehbarer Duschstuhl',
  },
  // ... resto de idiomas
};
\`\`\`

## Como Funciona la Traduccion

\`\`\`
Usuario visita /de/ (Aleman)
        |
        v
Sistema detecta locale = "de"
        |
        v
Busca producto.handle en PRODUCT_NAMES[de]
        |
        v
Muestra: "Premium Klapprollator" en vez de "Premium Folding Walker"
\`\`\`
